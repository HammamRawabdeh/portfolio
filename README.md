## 🚀 Tech Stack

* **HTML5** for semantic structure
* **CSS3** for styling and responsive layouts
* **Vanilla JavaScript** for interactive UI components

## 🎯 Features

* **Project Showcase:** Highlights my work across full-stack, backend, and frontend development.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
